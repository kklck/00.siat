#소켓 연결
1. 브라우저(클라이언트)가 서버에게 소켓 연결 요청(HandleOpen)
2. 서버가 중간처리 후, 해당 소켓 찾아 응답.
* 중간처리 : 서버 내 세션이라는 브라우저별 통신 객체를 각각 만들어 소켓 탐색.
3. 세션을 파라미터로 받은 소켓이 전체 세션에 해당 세션을 저장

#소켓 메시지
1. 브라우저가 메시지를 WebSocket.send() 이용하여 소켓에 전달.
2. 중간처리
* 중간처리 : 서버 내 세션이라는 브라우저별 통신 객체를 각각 만들어 소켓 탐색.
3. 세션과 메시지를 파라미터로 받은 소켓이 
3. 소켓을 요청한 세션을 제외한 모든 세션에게 메시지를 보냄.
5. 세션 내 sendText() 이용하여 각각 브라우저에 전송.
6. 브라우저가 onmessage(메시지내용)으로 받아 data출력
