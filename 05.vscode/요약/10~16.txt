- margin
    - 픽셀 값을 고정하더라도, 좌우로 남은 공간을 채움
    - 위-아래 마진 간 공간은 공유 가능. 픽셀이 가장 큰 값 만큼 공간 존재.
    - 죄-우 마진 간 공간은 공유 불가. 베타적. 각 픽셀 값을 더한 만큼 공간 존재.

- border-radius
    - 반지름 값 만큼 모서리 곡화.
    - 11시 방향부터 시계 방향으로 픽셀 값 지정.

- 다양한 스타일 값 한 번에
    - 선택자{ 스타일 속성 : 속성 값1, 속성 값2, ....}
    - 속성에 따라 값의 중복이 거의 없기 때문

- background-image
    - CSS3 이후로 배경 url 여러 개 선택 가능.
    - url('~.png', url('~~.png'))

- line-height
    - div의 height = a의 line-height

- 스타일 명
    - class="스타일명1 스타일명2 스타일명3...."

- 위치 속성
    - position
        - static : default, 위 아래로 순서 배치
        - relative : 작성한 위치 이동 값으로 상대 위치 지정.
        - absolute : 작성한 위치 이동 값으로 절대 위치 지정.
            - 기준 : absolute relative 으로 지정된 부모 기준, 없다면 좌측 상단.
        - 겹쳐 쓰고 싶을 때 <-> float : 둥둥 띄어 따로

- 가로 배치
    - 형식은 테이블이 유용하지만, 테이블은 배치에 쓰이지 않음
    - float로 가로형식의 배치 사용
    - float 한 개체가 둘 이상이면, 평면화 -> 화면 크기에 넘치면 아래로 내려가는 반응형.

- margin-left : 왼쪽 마진 공간 띄기
- padding-left : 왼쪽 패딩 공간(배경) 채우기

- clear : 일그러진 레이아웃 초기화
